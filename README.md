IntellectMap
Visualizing campus intelligence through AI-powered interest mapping.

Problem
Students struggle to discover relevant peers, clubs, and events.
Networking often favors seniors and existing connections.
There is no visual, interest-based discovery platform.

Solution
IntellectMap is an interactive campus discovery platform that:
Visualizes people, clubs, and events on a global/campus map

Uses TF-IDF similarity scoring for recommendations

Groups nodes by interest domains

Generates smart intro messages

Allows saving connections

How It Works
User selects interests and filters

Text data is vectorized using TF-IDF

Cosine similarity ranks relevant nodes

Interactive map visualizes results

Personalized recommendations are displayed

Tech Stack
Python

Streamlit

Plotly

Scikit-learn

Pandas / NumPy
